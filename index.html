Is this safe?

We utilize blunted-edge technologies to ensure your personal information is relatively safe. The files themselves are hosted on bluehost.com and the database runs from parse.com. The latter is also where all the security features are implemented. Parse was aquired by Facebook, so it is pretty relaible. Even I can not see your password. In additition to that access is restricter by ACL's and CLP's. However, it is entirely possible that I messed something up and there is a way to circumvent security. In fact, there probably is. The nature of the data we store is not very sensitive, however, so there is no need for concern.
